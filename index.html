Creare un progetto con il Web Framework AngularJs 



 
   
                           Per verificare versione sw installato  digitare :
                            - bower  --version  
                            - karma  --version 
                              
                            - creare directory dove inserie file es. ProgettoA poi digitare :
                            - cd ProgettoA
                            - git init
      - yo angular –minsafe –karma  (il prametro minsafe è necessario per essere minificato)
      - bower install  (modifica tutti i file necessari)
      - npm install    (installa tutte le dipendenze di js)
      - karma init     (chiede i parametri x test es. phantomjs)
         
      - il file karma.conf.js contiene i parametri di configurazione 
      - il comando  :  grunt karma       ci permette di verificare se tutto ok  
 
      - il comando  :  grunt server      attiva il web server (localhost:9000 )   
 A questo punto dobbiamo creare 
   il controler e la views :
   - yo angular:controller nome_controller
       es. ProgettoACtrl
   - yo angular:view  nome_view 
       es. ProgettoAView 
           
  
  
   - git status    per verificare i file creati 
           nel repository locale
 
   - Creare nella cartella del progetto es. PreogettoA 
           la sub directory  /app/data  
     ora in  c:ProgettoA/app/data  inserire il file json 
           dove ci sono i dati pre l'applicazione 
       
      Preparare ambiente di lavoro 
      -  npm install –g nodejs
      -  npm install –g bower
      -  npm install –g grunt-cli
      -  npm install –g yo 
      -  npm install –g generator-angular
      -  npm install –g karma
      -  Install phantomjs(browser dedicato ai test)
      
  Ci siamo ..... ora con un editor puoi modificare le views
    e i controller per il tuo  
      Front-end Framework con le direttive di AngularJs :
             - ng-app
             - ng-controller
             - ng-model
             - ng-repeat
             - ng-show
             - ed inoltre i servizi + filtri ecc ....
